Privacy Policy – SEO Checker Extension

Last updated: 2025-09-06

Your privacy is important. The SEO Checker extension does not collect any personal information and does not send any data to external servers. All processing is done locally in your browser.

Data Processed

The extension only analyzes the web pages you visit to provide SEO functionality, including:

  • Page title and meta description
  • Headings (H1, H2, H3)
  • Images and ALT attributes
  • Internal and external links
  • Word count and visible text
  • JSON-LD data present on the page

Purpose

The data is used exclusively locally in the browser to generate SEO reports. No data is stored or shared with third parties.

Data Sharing

No data is shared with third parties.

User Rights

Since no personal data is collected, no action is required by the user.
